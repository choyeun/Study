# 말과 글은 생각의 표현이며 생각의 결과이다

언어 - 표현 - 관계
사고 - 언어

단순히 빨리빨리는 언어를 잊게 한다

> [!note]
> 나의 언어의 한계는 나의 세계의 한계다

# 생각이 막연하고 불분명할 때 우리의 말과 글이 애매하고 횡설수설하기 쉽듯이 불명확한 언어 구조는 우리의 생각을 뚜렷이 세워 놓지 못한다

생각의 결과로 표현

확정 확장 수정 정리 축소

# 우리가 말하기와 글쓰기를 배우는 것은 방송인이나 웅변가, 작가나 기자가 되기 위해서가 아니다

표현이 중요하다

# 사고의 과정은 복잡다단하다

한번에 잘쓸수는 없다
글쓰기는 사고의 과정이다

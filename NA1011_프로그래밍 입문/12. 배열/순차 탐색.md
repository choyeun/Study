# 순차 탐색

순차 탐색은 배열의 원소를 순서대로 하나씩 꺼내서 탐색키와 비교하여 원하는 값을 찾아 가는 방법

```c
#include <stdio.h>

#define SIZE 10

int main(void) {
    int key, i;
    int list[SIZE] = {1, 2, 3, 4, 5, 6, 7, 8, 9, 3};

    printf("탐색할 값을 입력하세요: ");
    scanf("%d", &key);

    for (i = 0; i < SIZE; i++) {
        if (list[i] == key) {
            printf("탐색 성공 인덱스 = %d\n", i);
            printf("탐색 종료\n");
            return 0;
        }
    }

    printf("탐색 실패\n");
    return 0;
}

```

O(n)

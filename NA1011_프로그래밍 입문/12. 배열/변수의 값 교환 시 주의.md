# 변수의 값 교환 시 주의

```c
grade[i] = grade[least]; // grade[i]의 기존 값은파괴된다
grade[least] = grade[i];

// 올바른 방법

temp = list[i];
list[i] = list[least];
list[least] = temp;


// 흑마법
list[i] ^= list[least]
list[least] ^= list[i]
list[i] ^= list[least]
```

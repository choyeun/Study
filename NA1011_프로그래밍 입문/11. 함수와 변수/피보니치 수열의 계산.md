# 피보니치 수열의 계산

순환 호출을 사용하면 비효율적인 예

```c
int fib (int n){
if (n == 0) return 0;
if (n == 1) return 1;
return (fib(n-1) + fib(n-2));
}
```

순환 호출을 사용했을 경우의 비효율성
 같은항이 중복해서 계산됨
 예를 들어 fib(6) 을 호출하게 되면 fib(3) 이 4 번이나 중복되어서 계산됨
 이러한 현상은 n 이 커지면 더 심해짐

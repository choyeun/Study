# 생존 기간

정적 할당 (static allocation):
 프로그램 실행 시간 동안 계속 유지

자동 할당 (automatic allocation):
 블록에 들어갈 때 생성
 블록에서 나올 때 소멸 생존 기간을 결정하는 요인
 변수가 선언된 위치
 저장 유형 지정자

저장 유형 지정자
 auto
 register
 static
 extern

왼쪽에 있는 변수에 오른쪽의 수식의 값을 계산하여 대입

| 복합 대입 연산자 | 의미         |
| --------- | ---------- |
| x += y    | x = x + y  |
| x -= y    | x = x - y  |
| x *= y    | x = x * y  |
| x /= y    | x = x / y  |
| x %= y    | x = x % y  |
| x &= y    | x = x & y  |
| x \|= y   | x = x \| y |
| x ^= y    | x = x ^ y  |
| x >>= y   | x = x >> y |
| x <<= y   | x = x << y |

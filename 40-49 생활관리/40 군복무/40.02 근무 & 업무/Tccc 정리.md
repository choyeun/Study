# 전술적 현장 처치

안전 구역에 도착하면 ( 분대장 ) 처치자 계속 처치 지시, 나머지는 경계 명령

## M 대량출혈

1. 지남력 체크 : 시공간 구분 ex) who, where
2. 무장해제 시키고 일회용 장갑 착용 -> 출혈 부위 피복 제거 후 상처 부위 5 ~ 7 cm 위로 지혈대 적용 ( 적용하면서 동맥 확인 ) -> 2차 지혈대 적용 시간 적고 2차 지혈대 재평가 ( 1. 3 손가락 들어가는지 2. 출혈 지속되는지 3. 5~10초 요골 맥박 확인 ) -> 순환 평가 -> 목, 서혜부, 겨드랑이, 다리 쓸면서 추가 출혈 확인 -> if 출혈 피복제거 -> 1. 직접 압박 실시 -> ( 환자의 손을 이용해 출혈을 막고 무릎으로 눌러줌 ) 2. 손가락에 응급 지혈 거즈를 둘둘 감싸고 상처 부위에 넣고 3분간 직접 압박 실시 3. 개인 압박 붕대 마무리

## A 기도 관리

머리 기울림 턱 들어올리기 If 경추 손상 턱 밀어올리기 -> 기도 확보 후 입을 벌려 이물질 확인 후 손가락을 집게 모양으로 해서 이물질을 꺼내고 말려있는 혀 꺼내기 ( 그르렁 소리? -> 재확인 )

## R 호흡 관리

1. 보고, 듣고, 느끼며 15초간 호흡 확인 ( 호흡수 * 4 1분에 6회미만 or 20회 초과 확인 ) 가슴, 듣기, 빰 확인
2. 호흡 이상이 확인되면 상의 제거
3. 상처 발견 시
	1. 직접 압박 실시 ( 처치자 손 )
	2. 3면 밀폐 드레싱 적용
	3. 외측 부분의 한면 피가 흐르게 1/2 개방
	4. 통나무 굴리기 법으로 추가 사출구 확인
	5. 보고, 듣고, 느끼기 확인 -> 문제가 생기면 상처 찾기

## C 순환 평가

1. 지혈대 재평가 : 손가락 3개, 출혈 지속?, 5~10초 동맥 확인
2. 응급 지혈 거즈 재평가 : 오염 or 출혈 심할시 교체
3. 3면 밀폐 드레싱 재확인
4. AVPU 확인 A 의식 확인 V 터치 반응 P 고통 반응 U 반응 없음

## H 저체온증

피부가 차갑고 축축, 맥박이 빠르고 약하다면 모포를 활용해서 환자의 몸을 감싼다
젖은 옷은 제거
휴대용 가방에서 들것을 꺼내며 통나무 굴리기법 or 모포 들어서 부상자 옮기기
들것 위에 부상자 장구류 & 무기 + 들것 가방 ( 반대 방향으로 말아서 평평하게 ) 들것 위에 끈들 제거
부상자 안심 시키면서 이동 ( 수평유지, 머리가 전방 ) 들것 들어 구령으로 다같이 들것 들기

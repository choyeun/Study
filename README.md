# Study Repository

이 저장소는 대학 강의 필기를 관리하기 위해 만들어졌습니다. 각 과목은 과목 코드와 이름으로 된 폴더로 구성되며, 각 폴더는 단원별로 구분된 Markdown 파일을 포함합니다.

## 커밋 메시지 규칙

 - 강의: 강의 필기 추가 및 수정
 - 과제: 과제 관련 내용 추가 및 수정
 - 프로젝트: 프로젝트 관련 변경 사항
 - 시험: 시험 준비 자료 추가 및 수정
 - 수정: 일반적인 수정
 - 업데이트: 기존 내용 업데이트
 - 버그: 버그 수정

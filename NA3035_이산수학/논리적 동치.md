# [[Logical equivalence]]
## logically equivalent (동치)
- two truth tables are identical

| p   | q   | ~p ∨ q | p -> q |
| --- | --- | ------ | ------ |
| T   | T   | T      | T      |
| T   | F   | F      | F      |
| F   | T   | T      | T      |
| F   | F   | T      | T      |
## converse (역)
- converse of p -> q is q -> p

| p   | q   | p -> q | q -> p |
| --- | --- | ------ | ------ |
| T   | T   | T      |        |
| T   | F   | F      |        |
| F   | T   | T      |        |
| F   | F   | T      |        |
## contrapositive (대우)
- contrapositive of the p -> q is ~q -> ~p
- logically equivalent 

| p   | q   | p- > q | ~q -> ~p |
| --- | --- | ------ | -------- |
| T   | T   | T      |          |
| T   | F   | F      |          |
| F   | T   | T      |          |
| F   | F   | T      |          |
